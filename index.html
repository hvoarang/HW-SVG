Теория к заданию

Мы изучили два свойства, чтобы задать цвет:

  • color для текста;
  • background-color для фона.

    • Но, в svg можно изменить цвет еще парой свойств:

  • fill для заливки внутри контура;
  • stroke для цвета самого контура.

Как правило, stroke используется в иконках довольно редко, а fill практически всегда. Обычно требуется чтобы цвет иконки наследовал цвет родительского элемента (например ссылки). При этом, цвет svg должен меняться вслед за изменением цвета текста родительского элемента (например при :hover). Для наследования цвета текста родителя используют свойство fill со значением currentColor

Спрайт - это один файл с множеством разных изображений внутри, как правило, небольших по размеру декоративных элементов (иконок). Спрайты бывают svg и png. Техники для создания svg и png спрайтов очень разные, но смысл один - упаковать множество изображений в один файл с целью ускорить загрузку сайта.

CSS ПЕРЕМЕННЫЕ

CSS переменные - это способ задать значение и затем вызвать это значение в нужном месте. Синтаксис присвоения значения - это два дефиса -- и затем название переменной. Синтаксис вызова значения - это служебное слово var и имя переменной в круглых скобках.

Например:

<div class="square-box"></div>

square-box {
--diameter: 40px;
width: var(--diameter);
height: var(--diameter);
background-color: orangered;
}

ЗНАЧЕНИЕ ПО УМОЛЧАНИЮ

<div class="circle-box"></div>

.circle-box {
width: var(--diameter, 40px);
height: var(--diameter, 40px);
border-radius: 50%;
background-color: orangered;
}

При использовании вызова var можно использовать еще одно значение, через запятую, которое будет применяться только в случае, если переменная не будет задана.

ОБЛАСТЬ ВИДИМОСТИ

Если объявить переменную внутри определенного селектора, то в других селекторах переменная будет недоступна. Если мы хотим сделать переменную доступную для всех селекторов, то такую переменную нужно объявить внутри специального селектора :root

Например

<div class="circle"></div>
<div class="square"></div>

:root {
--heat-color: orangered;
--cold-color: navy;
}

.circle {
--diameter: 40px;
width: var(--diameter);
height: var(--diameter);
border-radius: 50%;
background-color: var(--heat-color);
}

.square {
width: var(--diameter, 60px);
height: var(--diameter, 60px);
background-color: var(--cold-color);
}

Для svg спрайтов используется следующий прием. Селектору задается css переменная со значением currentColor, что означает текущий цвет родителя. Затем эта переменная переносится в спрайт и добавляется атрибуту fill вместе со значением по умолчанию. Например :

.icon {
--color: currentColor;
}
